Live Demo Repository Link

// Rainy_Days

Rainy Days is a responsive multi-page e-commerce website built entirely with pure HTML and CSS, based on a previously created Figma design.

The goal of this project was to translate a finished design into a fully functional website while following best practices for semantic markup, responsiveness, and accessibility. The site includes navigation between all pages defined in the original site architecture and adapts smoothly to different screen sizes without horizontal scrolling.

Flexbox and CSS Grid were used to create flexible layouts, while media queries ensure consistent behavior across devices. Accessibility and quality were key priorities: semantic HTML5 elements, proper alt text, sufficient color contrast, and WCAG-friendly structure were applied throughout the project. The codebase follows DRY principles, is cleanly formatted, and validated using HTML and accessibility validation tools.

The website is deployed on GitHub Pages and tested across major browsers and devices using Developer Tools.

// Project_GameHub //

Stack: HTML, CSS, JS, REST API

JavaScript-powered online game store consuming a public API

Live Demo Repository Link

// GameHub

GameHub is an interactive online game store built with JavaScript that consumes a public API to dynamically display products and manage a shopping cart.

The project focuses on working with asynchronous JavaScript, API integration, and client-side state management. Product data is fetched from the GameHub API and rendered dynamically on the homepage, with filtering options by category and genre. Each product has a dedicated detail page, allowing users to explore more information before adding items to the basket.

The shopping cart functionality allows users to add and remove products, view a cart summary on the checkout page, and complete the flow with an order confirmation screen. All data is handled dynamically, with no hardcoded product information.

Error handling and loading indicators were implemented to ensure a clear and user-friendly experience during API requests. Asynchronous operations are handled using async/await, and the interface was tested to ensure functionality and accessibility across pages.

This project demonstrates practical use of JavaScript for building interactive web applications, including API consumption, dynamic DOM manipulation, and basic e-commerce logic.
